# Nalam — Meditation Audio Files

Place your ambient/background audio files in this directory with the exact filenames listed below.

The app will work silently if files are missing — no crash, just no audio.

## Required Files

| Filename | Session | Recommended Sound |
|---|---|---|
| `anulom_vilom.mp3` | Anulom Vilom | Soft flute, nature sounds |
| `bhramari.mp3` | Bhramari | Humming tones, resonance music |
| `kapalbhati.mp3` | Kapalbhati | Drum beats, energising percussion |
| `sitali.mp3` | Sitali | Ocean waves, cool breeze |
| `mindfulness.mp3` | Mindfulness | Tibetan singing bowls, soft chimes |
| `yoga_nidra.mp3` | Yoga Nidra | Delta binaural beats, rain |

## Free Audio Sources

- **Freesound.org** — search for "meditation ambient", "tibetan bowl", "flute meditation"
- **Pixabay Audio** — royalty-free, no attribution required
- **Zapsplat.com** — free with free account
- **MyNoise.net** — export as MP3

## File Format

- Format: MP3 (preferred) or AAC
- Bitrate: 128kbps is sufficient (keeps APK size low)
- Duration: 5–15 minutes (the service loops the file automatically)
- Target total size: under 20MB for all 6 files combined
